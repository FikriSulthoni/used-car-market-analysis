# used-car-market-analysis

Data analysis portfolio project using Excel and Tableau to analyze used car prices in Indonesia


## Problem Statement

Used Car market involves multiple factors that influence pricing, such as mileage, additional features, car brand, location, and transmission type. This project aims to identify which factors have the strongest impact on used car prices in Indonesia.

Pasar mobil bekas dipengaruhi oleh berbagai faktor yang memengaruhi harga kendaraan, seperti jarak tempuh, fitur tambahan, merek mobil, lokasi, dan jenis transmisi.
Proyek ini bertujuan untuk mengidentifikasi faktor-faktor mana yang memiliki dampak paling kuat terhadap harga mobil bekas di Indonesia.

## Dataset Overview

Total records: 609 used car listings
Data type: Structured Tabular Data
Scope: Major cities in Indonesia (Jakarta, Malang, Bekasi, Depok, and sorrounding cities in java)

### Key attributes

Car brand, year of manufacture, mileage (km), location (city), transmission type, vehicle additional features (1 = available, 0 = not available), prices (rp)

## Tools and Skills

Microsoft Excel: Data understanding, data cleaning, data validation, and data preprocessing.
Tableau: Data visualization and interactive dashboard
Data Analysis: Exploratory data, aggregation, and comparison

This project foucsed on analytical thinking and insight generate rather than predictive modelling.

## Key Insight

- Vehicles with lower mileage tend to have significantly higher prices
- Automatic transmission type cars are priced higher on average compared to manual transmission
- BMW dominates the premium prices among listed brands
- Listings in Jakarte Selatan show the highest average prices
- Additional features contribute to increased vehicle value
